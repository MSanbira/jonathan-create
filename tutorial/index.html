📦 What is HTML?

HTML is like building blocks! It tells the computer what things are on a page.

  • 🏷️ Headings (titles)
  • 📝 Paragraphs (text)
  • 🖼️ Images
  • 🔘 Buttons

🎮 Try It! Add HTML Elements

👆 Click the buttons to add things here!

📝 The Code

<h1>This is a Heading</h1>

<p>This is a paragraph.</p>

<button>Click Me!</button>

<img src="cat.jpg">

👆 Tags wrap around content!
They tell the browser what things are.

🎨 What is CSS?

CSS is like paint and decoration! It makes things look pretty.

  • 🌈 Colors
  • 📏 Sizes
  • ✨ Effects
  • 📍 Position

🎮 Try It! Style This Box

Style Me! 🎁

📝 The Code

.box {
  background-color: gray;
  width: 150px;
  height: 150px;
  border-radius: 10px;
}

👆 CSS uses properties and values!
Change them to style elements.

⚡ What is JavaScript?

JavaScript makes things happen! It's the magic that makes websites interactive.

  • 🖱️ Clicks do things
  • 🔢 Math and counting
  • 🎲 Games and animations
  • 💬 Showing messages
